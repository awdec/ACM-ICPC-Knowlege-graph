## 🌈 核心特性

### 🤖 智能问答
- **中文支持**: 原生支持中文自然语言查询
- **意图识别**: 基于正则表达式的高效意图匹配
- **参数提取**: 自动提取查询中的关键信息

### 📊 图数据库
- **关系查询**: 高效处理复杂的知识关系查询
- **图遍历**: 支持多跳关系的深度查询
- **数据安全**: 参数化查询防止注入攻击

### 🌐 Web界面
- **响应式设计**: 适配不同设备屏幕
- **实时反馈**: 显示查询处理的完整过程
- **调试友好**: 提供详细的执行信息

### 🔧 可扩展性
- **模块化设计**: 清晰的职责分离便于维护
- **配置灵活**: 支持环境变量配置
- **插件友好**: 易于添加新的查询类型
